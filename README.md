# demo-service

This is just a small demo application showing a few things one can do in a
simple setup of Node.js & React.

## Development

`docker-compose up`
`cd backend && yarn run worker:dev`
`cd backend && yarn run server:dev`
`cd frontend && yarn run start`

## Running tests
`yarn run test:dev`
